Web Development

MIS Application (PT Batara Mulya Jaya)

2021

Projek ini merupakan projek ketiga saya menggunakan React Js. Projek Ini dikerjakan dengan kolaborasi saya sebagai Web Frontend Dev dan Perancang UI (Web dab Mobile App) bersama teman saya Alif Wicaksana Ramadhan sebagai Backend Developer dan Mobile Developer.

MIS Application (PT Batara Mulya Jaya) ini merupakan aplikasi manajemen informasi berbasis Web dan Mobile App. Web dan Mobile App ini merupakan bentuk digitalisasi perusahaan PT Batara Mulya Jaya dalam hal menajemen informasi pegawai.

Mobile App yang dibuat diperuntukkan bagi seluruh pegawai pada perusahaan tersebut, dimana pegawai akan membuat akun pada aplikasi tersebut yang selanjutnya pegawai diharuskan untuk mengisi berbagai informasi Data Pribadi. Setelah pengisian data pribadi pegawai juga dapat mengecek slip gaji dan mendownloadnya setelah akun tersebut telah di verifikasi oleh admin (melalui Web)

Website yang dibuat diperuntukkan bagi admin perusahaan terebut untuk mengelola user/pegawai seperti menghapus user, melihat detail data user untuk di verifikasi, mengubah pegawai menjadi admin untuk keperluan tertentu bagi perusahaan. Selain itu pada web dapat menyimpan data pegawai baik keseluruhan maupun perorangan dalam bentuk pdf. Untuk keperluan gaji juga terdapat fitur upload slip gaji yang nantinya dapat diterima oleh setiap pegawai dan dapat dilihat melalui aplikasi mobile.
