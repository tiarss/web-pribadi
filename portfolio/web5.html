Web Development

Web Catalog (Games and Movie)

2021

Projek ini merupakan projek kedua saya menggunakan React Js. Selain itu saya juga menggunakan Chakra UI sebagai React Component Library untuk Membuat tampilan antarmuka web ini.

Web Catalog ini berisi informasi informasi mengenai Movie dan Game. Selain itu Web ini juga dapat menambahkan Informasi Tambahan mengenai Game dan Movie dengan bergabung terlebih dahulu menjadi member dengan cara membuat akun. Setelah itu member dapat mengakses informasi Game dan Movie lebih lengkap seperti menambah informasi, mengedit, ataupun menghapus data informasi Game dan Menu di Menu Member. Data disimpan di dalam database melalui API
